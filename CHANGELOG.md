# Changelog

## 3.2.0

- 将原生响应弹窗重构为无标题栏桌面字幕浮层。
- 新增九宫格固定位置和 X/Y 像素偏移设置。
- 新增字幕宽高、字号、透明度和推理显示开关。
- 新增“预览流式字幕”管理接口和 UI 按钮。
- 字幕窗口保持置顶，但在 Windows 下不主动激活或抢占输入焦点。
- API 测试页默认发送 `stream: true`，通过浏览器 `ReadableStream` 实时解析 SSE。
- 测试页最终回答和完整响应在生成过程中持续更新。
- 明确区分真实流式请求与 `stream: false` 的一次性完整响应。

## 3.1.0

- 新增独立进程运行的原生桌面响应弹窗。
- 新增管理页“原生响应弹窗”开关。
- 新增完成后自动关闭时间设置，默认 30 秒，范围 1～3600 秒。
- 流式正文与推理内容在原生窗口中分离显示。
- 多个并发请求分别创建窗口。
- Windows 下使用非激活显示方式，避免主动抢占输入焦点。
- 关闭原生弹窗开关时同步关闭当前弹窗。
- 原生弹窗进程、队列或桌面环境异常不会阻塞 API。
- `启动WebUI.bat` 不再自动打开 Web 监视器。

## 3.0.0

- 项目名称调整为 **LLM Relay Desk**。
- 新增独立实时响应页 `/monitor/`。
- 新增 WebSocket 事件通道 `/ws/monitor`。
- 新增 OpenAI SSE 正文与推理内容解析。
- 新增 Ollama NDJSON 正文与思考内容解析。
- 新增非流式响应捕获和前端平滑流式展示。
- 新增多请求分流、请求状态、耗时、来源及请求 ID 展示。
- 新增最近 60 条内存历史和清空接口。
- 监视事件采用非阻塞有界队列，慢客户端不会阻塞 API。
- API 响应新增 `X-Relay-Request-ID` 响应头。
- 新增 Windows 独立监视窗口启动脚本。
